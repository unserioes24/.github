## 🌿 What we do

We run a **24/7 live cannabis grow stream** — and we build all the tech around it ourselves.

No off-the-shelf solutions. No black boxes. Every sensor reading, every data point, every tool on the site — custom-built, transparent, and open where possible.

| | Project | What it is |
|--|---------|-----------|
| 🌱 | **[unserioes24.de](https://unserioes24.de)** | The hub — live stream, grow diaries, calculators, blog & product comparisons |
| 📡 | **[growdata-hub](https://github.com/unserioes24/growdata-hub)** | Sensor data pipeline, grow metrics API and storage layer |
